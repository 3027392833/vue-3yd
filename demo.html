🏥 战救指挥可视化监控大屏

基于 Vue 3 + Element Plus 构建的专业军事医疗监控系统

🎯 项目概述

这是一个专为军事医疗救援场景设计的可视化监控大屏系统，采用现代化的前端技术栈，提供实时数据展示、地图可视化、预警提醒等功能。

🚀 启动应用 📁 查看源码

✨ 核心功能

📊 全域医疗力量监控

实时展示建制卫勤力量、机动卫勤力量、野战医院、后方医院等关键医疗资源分布情况

📦 卫勤物资情况管理

监控关键物资消耗量、库存水平、消耗速率，支持预警和补充提醒

🗺️ 战场情况可视化

集成高德地图，展示医院位置、后送路径、实时警示信息

📈 战救效率统计

分析检伤分类时间、手术等待时间、后送周转时间等关键指标

🚑 医疗后送能力

监控运力数量、医护力量、实时状态，提供预警功能

🔄 实时数据更新

支持30秒自动更新，WebSocket实时推送，确保数据时效性

🛠️ 技术栈

Vue 3 Element Plus 高德地图 API Vite CSS3 JavaScript ES6+ Flex 布局 响应式设计

🎨 界面特色

🎯 专业军事风格

深蓝渐变背景，毛玻璃效果，专业军事监控界面设计

支持1920x1080、1600x900、1366x768等多种分辨率

🚨 预警系统

红色警告标识，关键信息高亮显示

🚀 快速开始

1. 环境要求

Node.js >= 16.0.0
npm >= 8.0.0

2. 安装依赖

npm install

3. 启动开发服务器

npm run dev

4. 访问应用

http://localhost:5173

⚙️ 配置说明

高德地图API配置

在 src/App.vue 中配置您的高德地图API密钥和安全码：

// 高德安全码配置
window._AMapSecurityConfig = {
  securityJsCode: "您的安全码"
}

// 地图API密钥
key: "您的地图API密钥"

数据接口配置

在 src/services/dataService.js 中配置后端API接口：

export const getMedicalForces = async () => {
  const response = await fetch('/api/medical-forces')
  return response.json()
}
